
# 자동화 업무 선정

## 정보 수집 - 논문 정리

## 업무 분석

### 1. 업무명
- 논문 분석 보고서 작성

### 2. 현재 수행 방식
- 소요 시간: 매주 금요일 60분
- 반복 주기: 매주 1회
- 수작업 내용: 논문 네이밍, 논문 분류, 논문 PDF 파일 markdown으로 변환, 변환된 markdown 파일 분석 요청

### 3. 입력 정보
- 분석하고자 하는 논문 파일명
- 용도(단순 내용 파악, 내용 분석)

### 4. 출력 결과물
- 분석 대상 논문 md파일
- 분석 대상 논문 분석 보고서 md파일

### 5. 자동화 가능 부분
- PDF 파일을 md 파일로 변환
- 변환한 md 파일 후처리
- 형식 통일 및 정리
- 분석 결과 도출

### 6. 사람이 해야 할 부분
- 최종 검토 및 수정
