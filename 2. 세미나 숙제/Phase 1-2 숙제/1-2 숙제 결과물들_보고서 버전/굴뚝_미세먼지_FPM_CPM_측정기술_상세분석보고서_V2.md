# 굴뚝 배출 미세먼지 FPM/CPM 측정 기술 상세 분석 보고서

**참고 문헌:** Rethinking primary particulate matter: Integrating filterable and condensable particulate matter in measurement and analysis

---

## 1. 논문 핵심 요약 (3문장)

1. **이중 희석 시스템(Doubled-dilution system) 개발:** 1단계 고온 희석(120°C)과 2단계 저온 희석(30°C)을 결합하여, FPM을 CPM 형성 경로에서 제외하지 않으면서 FPM과 1차 PM을 동시에 실시간 측정하는 시스템을 제안함.

2. **이종핵 형성(Heterogeneous nucleation)의 우세성 입증:** 필터 기반 질량 농도 측정 결과, 이종핵 형성(0.97 mg/m³)이 동종핵 형성(0.055 mg/m³)보다 **17.6배 높음**을 정량적으로 입증함.

3. **대기오염 저감장치(APCD)의 역설적 영향:** SCR에서 NH₄NO₃, (NH₄)₂SO₄, FGD에서 CaSO₄·2H₂O 등 2차 무기 에어로졸(SIA) 유사 화합물이 부산물로 생성되어 1차 PM 배출량을 증가시킴.

### 2.2 시스템 구성 및 흐름

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         이중 희석 시스템 상세 흐름도                           │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  [굴뚝 내부]        [가열 샘플링 라인]       [1단계 희석]        [2단계 희석]   │
│       │                   │                    │                   │        │
│   PM₂.₅ 사이클론  ──→  가열 유지  ──→     고온 희석     ──→    저온 희석      │
│   (등속 흡인)         (응축 방지)        (FPM 분리)        (CPM 형성)        │
│       │                   │                    │                   │        │
│   굴뚝온도            샘플링라인              1단계               2단계        │
│   115±3°C            115°C 유지            120°C               30±2°C       │
│                                                │                   │        │
│                                           ┌────┴────┐         ┌────┴────┐   │
│                                           │         │         │         │   │
│                                       HT-ELPI+   필터      ELPI+     필터   │
│                                       (120°C)   (FPM)    (30°C)   (1차PM)  │
│                                                                             │
│  ※ 체류 챔버: 2단계 샘플링 전단에 위치 (체류시간 10초)                          │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 2.3 각 단계별 측정 원리

#### 2.3.1 1단계: 고온 희석 (Hot Dilution) - FPM 측정

| 항목           | 설정값              | 원리                     |
| ------------ | ---------------- | ---------------------- |
| **희석 공기 온도** | **120°C**        | 굴뚝 온도(115±3°C)보다 높게 유지 |
| **희석 배율**    | **5배**           | 가스상 물질의 분압 저하          |
| **측정 장비**    | HT-ELPI+ (120°C) | 고온에서 고체 입자만 측정         |
| **목적**       | 휘발성 물질 기체상 유지    | 재핵형성(re-nucleation) 억제 |

**열희석(Thermodilution) 원리:**
- 휘발성 물질을 가열하여 기체상으로 유지
- 포화증기압 이하로 희석하여 응축 방지
- Particle Measurement Programme (2007년 도입) 기반

#### 2.3.2 2단계: 저온 희석 (Cool Dilution) - 1차 PM 측정

| 항목              | 설정값          | 원리                |
| --------------- | ------------ | ----------------- |
| **희석 공기 온도**    | **30±2°C**   | 대기 조건과 유사하게 냉각    |
| **희석 배율**       | **5배**       | 총 희석 배율 25배 (5×5) |
| **상대습도 (희석 전)** | **<70%**     | 수분 응축 제어          |
| **상대습도 (희석 후)** | **<10%**     | 건조 조건 유지          |
| **체류시간**        | **10초**      | CPM 형성 시간 확보      |
| **측정 장비**       | ELPI+ (30°C) | 상온에서 1차 PM 측정     |

## 3. 측정 결과에 영향을 주는 주요 변수 (구체적 수치 포함)

### 3.1 희석 시스템 운전 조건

| 변수 | 위치 | 본 연구 설정값 | 영향 |
|------|------|----------------|------|
| **1단계 희석 공기 온도** | 1단계 희석기 | **120°C** | ↓감소 시: 조기 응축 → FPM 과대평가 |
| **2단계 희석 공기 온도** | 2단계 희석기 | **30±2°C** | ↑증가 시: CPM 형성 억제 → 1차 PM 과소평가 |
| **1단계 희석 배율** | 1단계 희석기 | **5배** | ↑증가 시: 분압 저하 → CPM 형성 억제 |
| **2단계 희석 배율** | 2단계 희석기 | **5배** | ↑증가 시: 과포화 억제 → CPM 감소 |
| **총 희석 배율** | 전체 시스템 | **25배 (5×5)** | 현장 조건에 따라 조절 가능 |
| **샘플링 유량** | 각 장비 | **10 LPM** | 등속 흡인 조건 유지 필요 |

### 3.2 체류 챔버 환경 조건 (상세)

| 항목 | 설정값 | 비고 |
|------|--------|------|
| **챔버 형상** | 원통형 | 맞춤 설계 |
| **챔버 직경** | **100 mm** | - |
| **공기 유량** | **10 L/min** | ELPI+ 샘플링 유량과 동일 |
| **챔버 내 온도** | **30°C** | 일정하게 통제 |
| **레이놀즈 수 (Re)** | **132.8** | **층류 조건 (Re < 2300)** |
| **체류시간** | **10초** | 유량과 챔버 체적으로 결정 |

> **중요:** 체류 챔버는 **층류(laminar flow)** 조건에서 운전되며, 온도(30°C)와 유량(10 L/min)이 **일정하게 통제**된 상태임. 이는 CPM 형성의 재현성을 확보하기 위한 필수 조건임.

## 4. 변수 변화에 따른 결과값 변화

### 4.1 현장 실험 결과 (870 MW 석탄화력발전소)

![[Fig. 2.jpg]]
**Fig. 2.** (a) 실시간 입도분포 측정, (b) 필터 기반 질량 농도 비교

#### 4.1.1 입자 크기 분포 변화

| 측정 항목 | FPM | 1차 PM | 변화 |
|-----------|-----|--------|------|
| **핵형성 모드 크기** | **8 nm** | **18 nm** | +10 nm (125% 증가) |
| **축적 모드 크기** | **0.16 µm** | **0.18 µm** | +0.02 µm (12.5% 증가) |
| **입자 수 농도** | 낮음 | **현저히 높음** | 이종핵 형성 기여 |

#### 4.1.2 질량 농도 비교 (핵심 결과)

| 형성 메커니즘 | 측정 방법 | 질량 농도 | 비율 |
|---------------|-----------|-----------|------|
| **동종핵 형성** | 인스택 필터 사용 (CPM) | **0.055 mg/m³** | 1 |
| **이종핵 형성** | 인스택 필터 미사용 (1차 PM - FPM) | **0.97 mg/m³** | **17.6** |

> **핵심 발견:** FPM을 제외하고 측정할 경우, 실제 1차 PM 질량 농도가 **최대 65.3% 과소평가**됨

### 4.2 1차/2차 희석 조건 변화의 영향 (이론적 분석)

| 변수 변화 | 예상 결과 | 메커니즘 |
|-----------|-----------|----------|
| **1단계 온도 ↓ (120°C 미만)** | FPM 과대평가 | 휘발성 물질 조기 응축 |
| **1단계 온도 ↑ (120°C 초과)** | 측정 안정성 유지 | 단, 에너지 소비 증가 |
| **2단계 온도 ↓ (30°C 미만)** | 1차 PM 증가 가능 | CPM 응축 촉진 |
| **2단계 온도 ↑ (30°C 초과)** | 1차 PM 감소 | CPM 형성 억제 |
| **희석 배율 ↑** | CPM 감소 | 과포화비 감소 |
| **체류시간 ↑ (10초 초과)** | 입자 성장 증가 | 응축/응집 시간 증가 |

> **참고:** 본 논문은 위 조건(1단계 120°C, 2단계 30°C, 희석배율 5×5, 체류시간 10초)에서의 결과만 보고하며, **변수 변화에 따른 민감도 분석은 수행되지 않음**. 향후 연구에서 파라미터 최적화 연구가 필요함.

## 5. 체류시간에 따른 양상

### 5.1 체류 챔버의 역할

```
┌────────────────────────────────────────────────────────────┐
│                    체류 챔버 내 CPM 형성 과정                 │
├────────────────────────────────────────────────────────────┤
│                                                            │
│   [유입]                [체류 챔버]                [유출]    │
│     │                      │                       │       │
│  FPM + 가스상  ──→   ┌─────────────┐  ──→   1차 PM        │
│  CPM 전구체          │  체류시간    │        (FPM + CPM)   │
│                      │   10초      │                      │
│                      │             │                      │
│                      │ • 동종핵 형성 │                      │
│                      │ • 이종핵 형성 │                      │
│                      │ • 응집/성장  │                      │
│                      └─────────────┘                      │
│                                                            │
│   환경조건: 30°C, 층류(Re=132.8), 유량 10 L/min             │
└────────────────────────────────────────────────────────────┘
```

### 5.2 체류시간 10초의 의미

| 현상 | 체류시간 내 발생 | 결과 |

| **이종핵 형성** | FPM 표면에 CPM 응축 | 0.18 µm로 성장 |
| **입자 응집** | 소형 입자 간 결합 | 이중모드 분포 형성 |

## 7. 결론 및 연구 한계

### 7.1 핵심 결론

1. **이중 희석 시스템의 유효성:** 1단계 120°C, 2단계 30°C 조건에서 FPM과 1차 PM의 동시 측정 가능
2. **이종핵 형성의 우세:** 질량 기준 17.6배 높은 형성률
3. **APCD 부산물 기여:** SCR, FGD에서 SIA 유사 화합물 생성

### 7.2 본 논문의 한계 (향후 연구 필요)

| 항목              | 본 논문 현황    | 향후 연구 방향      |
| --------------- | ---------- | ------------- |
| **희석 온도 변화 영향** | 단일 조건만 테스트 | 온도 범위별 민감도 분석 |
| **희석 배율 변화 영향** | 5×5배 고정    | 최적 희석 배율 도출   |
| **체류시간 변화 영향**  | 10초 고정     | 체류시간별 CPM 형성률 |
| **다양한 배출원**     | 석탄화력 1개소   | 다양한 산업시설 적용   |
