## 3. 추측/추론 (논문에 없으며 계산 또는 추론한 것) - 신뢰도 **하**

| 항목                       | 보고서 기재 값 | 추측 근거                                |
| ------------------------ | -------- | ------------------------------------ |
| **굴뚝 샘플링 유량**            | ~0.4 LPM | 희석 배율(25배) 역산: 10 LPM ÷ 25 = 0.4 LPM |
| **1단계 희석 공기 유량**         | ~1.6 LPM | 5배 희석 가정: 2 - 0.4 = 1.6 LPM          |
| **2단계 희석 공기 유량**         | ~8 LPM   | 5배 희석 가정: 10 - 2 = 8 LPM             |
| **2단계 희석 전 유량**          | ~2 LPM   | 10 LPM ÷ 5 = 2 LPM                   |
| 1단계 온도 ↓ → FPM 과대평가      | -        | 물리화학적 원리 기반 추론 (논문 미검증)              |
| 2단계 온도 ↑ → Primary PM 감소 | -        | 물리화학적 원리 기반 추론 (논문 미검증)              |
| 희석 배율 ↑ → CPM 감소         | -        | 물리화학적 원리 기반 추론 (논문 미검증)              |
| 체류시간 ↑ → 입자 크기 증가        | -        | 물리화학적 원리 기반 추론 (논문 미검증)              |
| 상대습도 ↑ → 수용성 CPM 증가      | -        | 물리화학적 원리 기반 추론 (논문 미검증)              |

## 주의 사항

> [!warning] 유량 정보의 한계
> 보고서의 "유량 흐름 추정도"에 기재된 **굴뚝 샘플링 유량(~0.4 LPM), 1단계 희석 공기 유량(~1.6 LPM), 2단계 희석 공기 유량(~8 LPM)은 모두 추정값**입니다. 논문에는 최종 샘플링 유량(10 LPM)과 희석 배율(5×5)만 명시되어 있어, 역산으로 계산한 것입니다.

> [!warning] 변수-결과 상관관계
> "변수 변화에 따른 결과값 상관관계" 표의 내용은 **논문에서 실험적으로 검증되지 않았습니다**. 단일 조건(120°C/30°C, 5×5배, 10초)에서만 실험이 수행되었으며, 변수 변화에 따른 민감도 분석은 향후 연구로 제안된 사항입니다.

## 연구과제 수행 시 고려사항

1. **추정값 검증 필요:** 유량 관련 추정값은 실제 시스템 설계 시 반드시 재계산 및 검증 필요
2. **민감도 분석 수행:** 온도, 희석 배율, 체류시간 변화에 따른 영향을 실험적으로 검증해야 함
3. **추가 문헌 조사:** 본 논문에서 제공하지 않는 정보는 관련 표준(ISO 25597, OTM-37) 및 타 문헌 참고 필요
