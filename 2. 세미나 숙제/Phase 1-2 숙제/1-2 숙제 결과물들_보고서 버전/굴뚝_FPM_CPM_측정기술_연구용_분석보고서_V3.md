# 굴뚝 배출 미세먼지(FPM/CPM/Primary PM) 측정 기술 분석 보고서

**참고 문헌:** Rethinking primary particulate matter: Integrating filterable and condensable particulate matter in measurement and analysis

---

## 1. 핵심 요약 (3문장)

1. **FPM 측정 원리:** 1단계 고온 희석(120°C)을 통해 응축성 물질을 기체상으로 유지하면서, 고온 전기식 저압 충돌기(HT-ELPI+, 120°C)로 **고체/비휘발성 입자(FPM)만 선택적으로 측정**함.

2. **Primary PM 측정 원리:** 2단계 저온 희석(30°C) 후 체류 챔버(10초)를 거쳐, FPM 존재 하에서 **동종핵 형성(새 입자 생성) + 이종핵 형성(FPM 표면 응축)**이 모두 발생한 총 입자(Primary PM)를 ELPI+로 측정함.

3. **CPM 산출 원리:** CPM은 직접 측정이 아닌 **Primary PM에서 FPM을 뺀 차이값**으로 산출하며, 본 연구에서 이종핵 형성(0.97 mg/m³)이 동종핵 형성(0.055 mg/m³)보다 **17.6배 우세**함을 입증함.

### 2.3 FPM, CPM, Primary PM 정의 및 측정 방법

| 구분 | 정의 | 형성 메커니즘 | 측정 방법 |
|------|------|---------------|-----------|
| **FPM** | 필터로 포집 가능한 고체/비휘발성 입자 | 직접 배출 (연소 생성물) | 1단계 고온 희석 후 HT-ELPI+ (120°C) |
| **CPM** | 냉각/희석 시 기체에서 응축되어 형성된 입자 | 동종핵 형성 (FPM 없이) | 인스택 필터로 FPM 제거 후 측정 |
| **Primary PM** | FPM + 모든 응축성 물질 (동종+이종핵) | 동종핵 + 이종핵 형성 | 2단계 저온 희석 후 ELPI+ (30°C) |

### 3.2 유량 흐름 추정 (희석 배율 기반 계산)

> **참고:** 논문에 각 단계별 유량이 명시되지 않아, 희석 배율과 최종 샘플링 유량(10 LPM)을 기반으로 추정함.

```
희석 배율 = (샘플 유량 + 희석 공기 유량) / 샘플 유량

[유량 흐름 추정도]

굴뚝 샘플링 ──→ 1단계 희석 ──→ 2단계 희석 ──→ ELPI+ 샘플링
    │              │              │              │
   ~2 LPM    +8 LPM (120°C)  +8 LPM (30°C)     10 LPM
    │         희석공기           희석공기          │
    │              │              │              │
    └──────────────┴──────────────┴──────────────┘
         5배 희석        5배 희석       총 25배

※ 추정 계산:
• ELPI+ 샘플링 유량: 10 LPM (논문 명시)
• 2단계 희석 전 유량: 10 ÷ 5 = 2 LPM
• 1단계 희석 전 유량: 2 ÷ 5 = 0.4 LPM (굴뚝 샘플링 유량 추정)
• 1단계 희석 공기 유량: 2 - 0.4 = 1.6 LPM (추정)
• 2단계 희석 공기 유량: 10 - 2 = 8 LPM (추정)
```

## 5. 결론

1. **이중 희석 시스템의 유효성 검증**
   - 1단계 고온 희석(120°C)과 2단계 저온 희석(30°C)의 조합으로 FPM과 Primary PM 동시 실시간 측정 가능
   - 입자 손실률 <5%로 시스템 안정성 확보

2. **이종핵 형성의 우세성 정량화**
   - 질량 기준 이종핵 형성이 동종핵 형성보다 **17.6배** 높음
   - 기존 방법(FPM 제외)으로 측정 시 **최대 65.3% 과소평가**

3. **APCD 부산물의 CPM 기여 확인**
   - SCR: NH₄NO₃, (NH₄)₂SO₄ 생성
   - FGD: CaSO₄·2H₂O 생성
