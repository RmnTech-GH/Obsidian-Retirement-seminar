# 굴뚝 배출 미세먼지 FPM/CPM 측정 기술 분석 보고서

**참고 문헌:** Rethinking primary particulate matter: Integrating filterable and condensable particulate matter in measurement and analysis

---

## 1. 논문 핵심 요약 (3문장)

1. **이중 희석 시스템(Doubled-dilution system) 개발:** FPM을 CPM 형성 경로에서 제외하지 않으면서 FPM과 1차 PM을 동시에 실시간 측정할 수 있는 이중 희석 시스템을 제안함.

2. **이종핵 형성(Heterogeneous nucleation)의 우세성 입증:** 필터 기반 질량 농도 측정 결과, 이종핵 형성이 동종핵 형성(Homogeneous nucleation)보다 17.6배 높은 것으로 나타남.

3. **대기오염 저감장치(APCD)의 역설적 영향:** SCR, FGD 등 대기오염 저감장치가 2차 무기 에어로졸(SIA) 유사 화합물을 부산물로 생성하여 오히려 1차 PM 배출에 부정적 영향을 미침.

## 2. 측정 원리

### 2.1 기존 측정 방법의 한계

| 측정 방법 | 원리 | 한계점 |
|-----------|------|--------|
| **EPA Method 202 (냉각법)** | 임핀저를 사용하여 배기가스를 냉각, CPM 응축 | - 동종핵 형성만 고려<br>- SO₂, HCl, NH₃ 등의 용해로 인한 양의 오차 (3.8~20.7%)<br>- 장비 표면 CPM 손실 (75%) |
| **ISO 희석법 (OTM-37)** | 배기가스를 희석하여 과포화 상태 유도 | - FPM 특성 분석 불가<br>- 1차 PM이 CPM으로 오분류됨 |

### 2.2 이중 희석 시스템의 원리

```
┌─────────────────────────────────────────────────────────────────────┐
│                    이중 희석 시스템 구성도                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   [굴뚝]     [가열 샘플링 라인]    [1단계 희석]     [2단계 희석]        │
│     │              │                  │               │             │
│  PM₂.₅  ──→   115±3°C   ──→    고온희석 120°C  ──→  저온희석 30°C    │
│  사이클론                          (FPM 샘플링)     (1차 PM 샘플링)    │
│                                        │               │             │
│                                   HT-ELPI+          ELPI+           │
│                                  (실시간 측정)     (실시간 측정)        │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

#### 2.2.1 1단계: 고온 희석 (Hot Dilution) - FPM 측정

- **온도:** 120°C (굴뚝 온도 115±3°C에 적응)
- **목적:** 휘발성 물질을 기체상으로 유지하면서 고체 입자(FPM)만 분리
- **원리:**
  - 가스상 물질의 분압을 낮춤
  - 포화증기압 이하로 희석하여 재핵형성 억제
  - 열희석(Thermodilution) 과정으로 정확한 고체 입자 측정
- **장비:** 고온 전기식 저압 충돌기(HT-ELPI+, 120°C 운전)

#### 2.2.2 2단계: 저온 희석 (Cool Dilution) - 1차 PM 측정

- **온도:** 30±2°C
- **상대습도:** 70% 이하 (희석 후 <10%)
- **체류시간:** 10초 (체류 챔버 사용)
- **목적:** FPM 존재 하에서 CPM 형성 → 1차 PM 측정
- **원리:**
  - 동종핵 형성 + 이종핵 형성 동시 발생
  - 응축성 물질이 기존 FPM 표면에 응축 (이종핵 형성)
  - 새로운 핵 생성 (동종핵 형성)

#### 2.2.3 핵심 개념 정의

| 구분 | 정의 | 측정 방법 |
|------|------|-----------|
| **FPM (Filterable PM)** | 필터로 포집 가능한 고체/액체 입자 | 고온 희석 단계에서 직접 측정 |
| **CPM (Condensable PM)** | FPM 제외 시 동종핵 형성으로 생성된 입자 | 인스택 필터 사용 (직접 희석법) |
| **1차 PM (Primary PM)** | FPM + 동종/이종핵 형성 CPM | 인스택 필터 미사용 (간접 희석법) |

### 2.3 분석 방법

| 분석 항목 | 장비/방법 | 측정 대상 |
|-----------|-----------|-----------|
| 실시간 입도분포 | ELPI+, HT-ELPI+ | 입자 수 농도, 크기 분포 |
| 형태 분석 | FE-SEM, HR-TEM | 입자 형태, 표면 구조 |
| 화학 조성 | Raman 분광법 | 개별 입자 화학 성분 |
| 무기 성분 | 이온 크로마토그래피 | 수용성 이온 정량 |

## 3. 측정 결과에 영향을 주는 주요 변수

### 3.1 희석 조건

| 변수 | 변화 | 결과값 영향 |
|------|------|-------------|
| **희석 온도** | ↑ 증가 | CPM 응축 억제 → FPM만 측정됨<br>↓ 감소 시 CPM 형성 증가 |
| **희석 배율** | ↑ 증가 | 가스 분압 감소 → 과포화 억제 → CPM 감소<br>↓ 감소 시 CPM 형성 증가 |
| **상대습도** | ↑ 증가 | 수용성 CPM 흡습 성장 → 질량 농도 증가 |
| **체류시간** | ↑ 증가 | 응축/응집 시간 증가 → 1차 PM 성장 |

### 3.2 배기가스 특성

| 변수 | 변화 | 결과값 영향 |
|------|------|-------------|
| **굴뚝 온도** | 115±3°C 기준 | 고온 희석 단계 온도 조절 필요 |
| **SO₂ 농도** | ↑ 증가 | CaSO₄·2H₂O, (NH₄)₂SO₄ 형성 증가 |
| **NH₃ 농도** | ↑ 증가 (SCR 과잉 주입) | NH₄NO₃, (NH₄)₂SO₄ 형성 증가 |
| **FPM 농도** | ↑ 증가 | 이종핵 형성 반응 표면 증가 → 1차 PM 증가 |

### 3.3 입자 형성 메커니즘에 따른 영향

```
┌────────────────────────────────────────────────────────────────────┐
│                     CPM 형성 메커니즘 비교                           │
├─────────────────────┬──────────────────────────────────────────────┤
│   동종핵 형성         │   이종핵 형성                                 │
│   (Homogeneous)     │   (Heterogeneous)                            │
├─────────────────────┼──────────────────────────────────────────────┤
│ • 높은 과포화비 필요   │ • 낮은 과포화비에서도 발생                      │
│ • 새로운 핵 생성       │ • 기존 FPM 표면에 응축                        │
│ • 핵형성 모드 (8 nm)   │ • 축적 모드 (0.18 µm)                        │
│ • 질량 농도: 0.055 mg/m³ │ • 질량 농도: 0.97 mg/m³                   │
│                     │ • **17.6배 높은 형성률**                       │
└─────────────────────┴──────────────────────────────────────────────┘
```

### 3.4 대기오염 저감장치(APCD) 부산물

| 장치 | 부산물 | 형성 반응 | 영향 |
|------|--------|-----------|------|
| **SCR** | NH₄NO₃ | 2NO₂ + 2NH₃ → NH₄NO₃ + N₂ + H₂O (50~150°C) | 1차 PM의 무기 성분 증가 |
| **SCR** | (NH₄)₂SO₄ | 2NH₃ + SO₃ + H₂O → (NH₄)₂SO₄ | FPM에서 검출 |
| **FGD** | CaSO₄·2H₂O | CaSO₃ + ½O₂ → CaSO₄ → CaSO₄·2H₂O | 1차 PM의 주요 성분 |

### 3.5 입자 크기별 화학 조성 분포

| 크기 범위 | FPM | 1차 PM |
|-----------|-----|--------|
| **0.2-1.0 µm** | 그래파이트 탄소 | 그래파이트 탄소, NaNO₃ |
| **1.0-2.0 µm** | (NH₄)₂SO₄, CaSO₄·2H₂O, 탄소 | CaSO₄·2H₂O, NH₄NO₃, 탄소 **(다양성 증가)** |
| **2.0-2.5 µm** | 미검출 | CaSO₄·2H₂O, KNO₃, NaNO₃ **(1차 PM에서만 검출)** |

## 4. 핵심 결론 및 시사점

1. **FPM 배제 시 과소평가:** FPM을 1차 PM 측정에서 제외할 경우, 실제 1차 PM 질량 농도가 **최대 65.3% 과소평가**됨

2. **이중 희석 시스템의 필요성:** 기존 방법(EPA 202, ISO 희석법)의 한계를 극복하고 실제 대기 중 응축 거동을 정확히 반영

3. **APCD 개선 필요성:** 기존 저감장치가 SIA 유사 화합물을 생성하므로, CPM 형성 최소화를 위한 저감장치 개선 연구 필요
