#### Claude 설치하고 난 후 Path 환경변수 설정이 필요함 - 환경변수 설정 방법은 아래와 같음
  ##### Windows 설정 화면에서 영구적으로 추가 (권장)
   1. **시작 메뉴**에서  
    `환경 변수` 또는 `environment variables` 검색
    
2. **“시스템 환경 변수 편집”** 열기
    
3. 아래 버튼 클릭  
    **환경 변수(N)...**
    
4. 위쪽 “사용자 변수(User variables)”에서  
    **Path** 선택 → **편집(Edit)...**
    
5. **새로 만들기(New)** 클릭 후 아래 경로 붙여넣기:
    
`C:\Users\leepr\.local\bin`

6. 확인(OK) 계속 눌러서 다 닫기
    
7. **PowerShell/터미널을 전부 닫고 다시 열기**

### git 저장소 초기화 명령어 의미 정리
1. commit은 변경사항을 하나의 버전으로 저장해둔 기록
2. 어떤 파일이 바뀌었는지, 무엇이 추가/수정되었는지/누가 했는지/언제/커밋 메세지/ID 등이 포함
3. 되돌리기, 히스토리 추적, 협업 등이 가능함
